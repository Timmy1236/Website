## Instalación
1. [Clona](https://docs.github.com/es/repositories/creating-and-managing-repositories/cloning-a-repository) el repositorio.
```bash
git clone https://github.com/Timmy1236/Website.git
```
2. Instala las dependencias de node.js, *no es necesario para el funcionamiento de la pagina, pero si para [11ty](https://www.11ty.dev/).*
```bash
npm install
```
3. Enciende un servidor para visualizar la pagina, puedes usar la extension [live server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) en **vscode**, o también puedes usar **python**.
```bash
cd public && python -m http.server 8000
```
