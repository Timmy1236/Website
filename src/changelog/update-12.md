### Información
Después de varios años de usar GitHub Pages para hostear mi pagina web, decidí cambiarme de host por la de NekoWeb, con la ayuda de "deploy2nekoweb" me permite seguir guardando el contenido de mi pagina en GitHub pero automáticamente subir cualquier tipo de actualización a NekoWeb a traves de su API. Si estas viendo este mensaje, significa que debería de haber funcionado. espero.
